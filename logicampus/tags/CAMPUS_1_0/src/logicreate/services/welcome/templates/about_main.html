Introduction

Welcome to the LogiCampus demo CD. This CD contains a fully functional demo of the LogiCampus system. LogiCampus is a new Open Source distance learning / course management product. The system comes with no up front cost or reocurring fees that come with other commercial systems.

This project was developed by Tap Internet in conjunction with Tarrant County College in Texas. The project was developed with public grant money, and accordingly is being released under a public license.

LogiCampus Demo

The LogiCampus demo can be ran from three different standpoints; administrator, teacher and student. Each standpoint has different options and different applications available to them. The username/password combinations are "admin/admin", "teacher/teacher" and "student/student".

You can launch the LogiCampus demo by clicking here.

About LogiCampus

LogiCampus is a new distance learning system built in conjunction with Tarrant County College Center for Distance Learning of Fort Worth, Tx. LogiCampus will allow a College, University, or School District to run existing classes or online learning classes via the web as well as provide the foundation for an institution to create additional integrated applications.

LogiCampus provides a single sign-on for students, faculty, and staff of an institution and provides more than just a course management / distance learning system. LogiCampus provides the standard tools for faculty to create their online courses, process assignments, make tests and stay in contact with students. Students enjoy being able to easily see their assignments, lessons and other information for all of their classes. Staff benefit by other applications such as the built-in help desk, textbook requisitions, exam scheduling and faculty / course changes.

LogiCampus is built on top of Tap Internets open source application server called LogiCreate, stated Keith Elder, Director of Tap Internet. Our learning system provides a foundation on which an institution can build additional integrated applications easily. Since the system is using LogiCreate, the sky is the limit as far as adding additional applications to meet departmental needs while still providing a single sign-on environment.

Tap Internet does not warrant that the CD will function for any other purpose. For security purposes networking and many other services have been disabled.
